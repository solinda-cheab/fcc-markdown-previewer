# Markdown Previewer

A live Markdown Previewer built with React and TypeScript for the FreeCodeCamp Front End Development Libraries Certification. The application allows users to write GitHub Flavored Markdown (GFM) and instantly preview the rendered HTML in real time.

##  Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fcc-markdown-previewer.git
```

Navigate to the project directory:

```bash
cd fcc-markdown-previewer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
